__This is the repository for all my resource packs that aren't Volcanic Dark, which you can find [here](https://github.com/TheBlondBoy/volcanic-dark).__ They'll be updated from time to time to add support for the latest Minecraft updates or add a new feature or two.

While the packs here can be downloaded and used, they may contain unfinished or in-development features.

Suggestions for mod support are welcome, but I can't guarantee I'll actually add support for 'em.
